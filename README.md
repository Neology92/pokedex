## Getting Started

Install all packages.

```bash
npm install
```

Simply run webpack dev server, and check running app on localhost:3000/

```bash
npm start
```

You can also build app and open it as index.html file in /dist folder

```bash
npm build
./dist/index.html
```

## Running the tests

```bash
npm test
```
